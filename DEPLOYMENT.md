PLATE HUNT — INSTALL ON IPHONE

This folder is the installable web app.

1. Upload every file in this folder to an HTTPS static website.
2. Open that website in Safari on the iPhone.
3. In Plate Hunt, open More > Install on iPhone.
4. In Safari, tap Share > Add to Home Screen.
5. Keep Open as Web App enabled and tap Add.

After the initial successful load, the app files are cached for offline use.
Trip data stays on the iPhone. Use Export Backup before clearing Safari data or changing phones.

SAFETY: A passenger should log plates while the car is moving. Drivers should wait until safely parked.
